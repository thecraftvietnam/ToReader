# User Guide — ToReader Manga Viewer

## 1. Installation
- Download `ToReader.exe` (already built).
- Place it anywhere (e.g., `D:\MangaViewer\`).
- Double-click `ToReader.exe` to launch.

## 2. Opening manga
- Click **📁 Manga Folder** to open a folder containing images or chapters.
- Click **📄 Manga File** to open a `.cbz` or `.pdf` file.
- Use the sidebar (`☰`) to switch chapters.

## 3. Navigation
- Scroll with the mouse wheel.
- Use **Prev Chapter** / **Next Chapter** buttons at the bottom.
- Zoom controls:
  - `＋` zoom in
  - `－` zoom out
  - `⊡ Fit Width` resets zoom

## 4. Notes
- Supported image formats: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`, `.webp`.
- `.cbz` archives and `.pdf` files are supported.
- If PDFs don’t open, ask the developer to ensure PyMuPDF was bundled.
