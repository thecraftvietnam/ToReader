TOREADER

ToReader is a free, non‑commercial desktop application designed for manga fans who want a smooth and flexible reading experience. It supports multiple formats and sources, making it easy to enjoy your favorite series in one place.

✨ Features

Website Integration: Browse and read directly from popular sites like Mangadex, Blogtruyen, and more.

Format Support: Open and read images, CBZ archives, and PDF files seamlessly.

Flexible Folder Selection: Choose a folder containing:
- Individual images
- Chapter folders
- Single CBZ files
- Single PDF files

User-Friendly Interface: Simple navigation and clean layout for distraction‑free reading.

Cross‑Platform Compatibility: Works with standard Windows .exe builds.

✨ Getting Started

Download the latest release from the repository.

1/ Run the .exe file on your Windows machine.

2. Select your preferred source (folder, CBZ, or PDF).

3/ Start reading instantly!
